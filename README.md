## 주요 기능

### 사용자 도메인
- 회원 가입
- 로그인
- 회원정보 수정
- 구매, 판매 내역 조회

### 채팅 도메인
- 실시간 채팅
- 푸시 알림[(알림 서비스 github)](https://github.com/taek105/AlarmServer)

### 상품 도메인
- 상품 등록, 수정, 삭제
- 상품 입찰 및 즉시 구매
- 상품명 검색
- 상품명 검색어 추천
